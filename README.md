# iLink Enterprise SDK

Enterprise-oriented Java SDK aligned to the documented iLink HTTP/JSON protocol.

Included:
- QR login + login long polling
- getupdates polling and latest context_token caching
- sendmessage text + image using latest context_token
- getconfig + sendtyping
- getuploadurl + CDN upload/download with AES-128-ECB media path
- unified config, retry, state, builder, listeners, executor manager
- heartbeat-style health probe for logged-in session
- slf4j logging

This package focuses on a strong enterprise SDK foundation and the core protocol chain.
